# CS-157 HopeBridge

HopeBridge is a platform that connects generosity directly to those in need.

## Project Structure
- `HopeBridge Backend Prototype/` - Spring Boot REST API
- `HopeBridge Frontend Prototype/` - Flutter mobile/web app
- `Marketing Website/` - Static marketing website
