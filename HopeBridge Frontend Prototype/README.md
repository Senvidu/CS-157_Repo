# HopeBridge Flutter Frontend

A Flutter mobile/web application for the HopeBridge platform.

## Tech Stack
- Flutter 3.41.5
- Dart
- GoRouter
- SharedPreferences
- HTTP package

## How to Run
1. Make sure backend is running on localhost:8080
2. Run: `flutter pub get`
3. Run: `flutter run`

## Features
- Donor login and signup
- Recipient login and signup
- Fundraising page
- Job finder
- eShop with voucher points
- Cart and checkout
